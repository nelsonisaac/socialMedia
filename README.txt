body-parser - is a npm module which process the data sent in HTTP request body

bcrypt - is a library which helps in hashing the passwords

cors - cross-origin Resourses Sharing, is a mechanism which helps in sharing resourses from the browser to the server in a secure manner

dotenv - helps in maintaning environment variables

gridfs-stream - easily helps in stream files to and from MongoDB gridfs

multer - Multer is a node.js middleware for handling multipart/form-data, which is primarily used for uploading files. It is written on top of busboy for maximum efficiency.

multer-gridfs-storage - GridFS storage engine for Multer to store uploaded files directly to MongoDb.


helmet - helps you secure your Node. js application by setting several HTTP headers. It acts as a middleware for Express and similar technologies, automatically adding or removing HTTP headers to comply with web security

morgan - for logging purposes

jsonwebtoken - An implementation of JSON Web Tokens.

mongoose - Mongoose is a MongoDB object modeling tool designed to work in an asynchronous environment.
